Platelet UI

Stackable, Tileable, Stretchy UI

Platelet is designed for both desktop and mobile devices. It provides a fluid layout without using media queries.

Platelet components have smooth transitoons and most elements respond to a user's actions. There has been relavent studies showing how flashing and quick moving / jumpy items on a web page are bad for the brain. (link)

Other UIs are way too ".classy", Platelet uses an element-first approach which revolutionizes the simplicity of web design. This lets developers create markup without having to look up class names, and it also makes web pages lighter and load quicker.
